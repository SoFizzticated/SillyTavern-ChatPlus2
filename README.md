# ChatPlus 2

Enhanced chat management extension for SillyTavern with improved architecture and stability.

## Installation

1. Place this folder in `SillyTavern/public/scripts/extensions/third-party/`
2. Restart SillyTavern or reload extensions
3. Enable "ChatPlus 2" in the Extensions menu
